# 🏫 LIL-CAMP – Lost Item Logger System for College Campus

**1st Year Even Sem Python Project**

---

## 📌 Overview
The Lost Item Logger System for College Campus (LIL-CAMP) is a Python-based application designed to manage and track lost and found items within a college environment. The system provides a simple menu-driven interface to perform operations such as adding, viewing, searching, and analyzing lost items efficiently.

---

## ⚙️ Features
- Add lost item details  
- View all stored items  
- Search items using keyword  
- Generate reports (Lost vs Found)  
- Graph visualization  
- JSON-based data storage  

---

## 🛠️ Tech Used
- Python  
- OOP + Inheritance  
- JSON File Handling  
- Pandas  
- NumPy  
- Matplotlib

## ▶️ Run
```bash
pip install pandas numpy matplotlib

```
---

🧩 Modules
- **Models Module** → Defines data structure using classes (BaseItem and Item with inheritance)  
- **Storage Module** → Handles JSON file operations using Pandas (load and save data)  
- **System Module** → Implements core functionalities like add, view, search, and report  
- **Main Module** → Controls program flow using a menu-driven interface
